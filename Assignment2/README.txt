INPUT FILE USED: inputFile_1.txt 2nd one Dr. Joshi put on canvas

CSX MACHINE USED: csx3.cs.okstate.edu


How to run program:  


1. Download the second input file that Dr. Joshi put on canvas.(inputFile_1.txt)

2. Take note of its file path, so that program can find it. 

3. Use putty to open csx3.cs.okstate.edu machine 

4. Download all .java files into desired directory

5. Run command: javac *.java

6. Run command: java Main 
    - Program should now be running with menu displayed

7. Enter 1, then hit enter key.

8. Enter the file path of .txt file to be read

9. Use other 8 options at your discretion
