CSX MACHINE USED: csx3.cs.okstate.edu

How to run program: 

1. Download the input file from canvas into the same directory as you plan to put the program (or just use the inputFile.txt in my repository)

2. Use putty to open csx3.cs.okstate.edu machine

3. Download all .java files and the mypackage directory into a the same directory as the .txt file from canvas

4. Run command: javac *.java to compile all files
    - Must do so in the myPackage directory as well

5. Run command: java Main 
    - You will then be prompted to enter the name of the file that you wish to read. 

6. The output should be displayed with all options outlined in the handout displayed.
